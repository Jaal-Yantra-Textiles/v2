# JYT Commerce Documentation

Welcome to the documentation for **JYT Commerce**, a Medusa 2.x-based textile commerce platform with integrated ad planning, social media publishing, analytics, and AI features.

## Who is this for?

### End Users & Partners

Start with **[Guides](/docs/guides/intro)** for step-by-step instructions on:

- Setting up social media publishing (Facebook, Instagram, X/Twitter)
- Configuring analytics tracking
- Deploying to Railway or Render
- Using the Ad Planning dashboard

### Development Team

Dive into **[Implementation](/docs/implementation/intro)** for:

- Architecture decisions and module designs
- Workflow builder internals
- AI/RAG integration details
- Security and encryption services

### API & Platform Reference

Check **[Reference](/docs/reference/intro)** for:

- Social media API schemas and analysis
- Platform-specific configurations (Facebook, Instagram, X)
- Webhook setup and troubleshooting
- UI component documentation
